Schedule-adjusted table and end-of-phase projections (UCL league phase)

This project’s main output is a schedule-adjusted table that reflects the different difficulty of opponents played so far.
We compare each team’s actual points against an expected points value for the difficulty of the matches (opponent level and venue (home or away))
Projections are projected end-of-phase totals based solely on the results of matches played so far. (A legitimite prognosis would require augmentation with additional information and interpretation).
